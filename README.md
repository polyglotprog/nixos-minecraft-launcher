# NixOS Minecraft Launcher Configuration (Prism Launcher)
Nix project that installs the open-source Minecraft [Prism Launcher].

1. Open a shell and change to this project directory.
2. Run `nix-shell`.
3. Run `prismlauncher` to start the launcher.
4. Select configuration options (UI customization, Java version, etc.)
5. Log in to your Microsoft account.
5. Click "Add Instance" and install the version of Minecraft you wish to play.
   Wait for the installation to complete.
6. Double click the Minecraft version and start playing!
